## User Story

**En tant que** membre du Bureau,
**Je veux** voir les derniers adhérents inscrits,
**Afin de** suivre l'activité récente.

---

## Critères d'acceptance

- [ ] Endpoint GET /dashboard/recent-members
- [ ] Paramètre : limit (default 10)
- [ ] Retourne : nom, prénom, tomité, date inscription

### Frontend

- [ ] Liste avec photo miniature
- [ ] Indication du tomité
- [ ] Date relative ("il y a 2h")

---

## Données retournées

```typescript
interface RecentMember {
  id: string;
  firstName: string;
  lastName: string;
  photoUrl: string | null;
  tomite: {
    code: string;
    name: string;
  };
  createdAt: string;
}
```

---

## Tâches techniques

### Backend

- [ ] Créer endpoint getRecentMembers
- [ ] Requête ORDER BY createdAt DESC LIMIT
- [ ] Inclure les infos du tomité (join)

### Frontend

- [ ] Créer composant RecentMembersList
- [ ] Afficher avatar avec initiales si pas de photo
- [ ] Formater la date en relatif (date-fns)
- [ ] Lien vers fiche adhérent
